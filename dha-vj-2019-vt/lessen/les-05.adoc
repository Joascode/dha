=== Samenvatting
Na introductie van TypeScript in de vorige les kijken we naar specifieke features van nieuwe ECMAScript versies ES6, ES7 en ES8, ook wel ES 2015, 2016 respectievelijk 2017 genoemd.
We kijken naar de achtergrond van de nieuwe versienummering, en hoe we kunnen achterhalen welke taal features nu al door (welke) browsers worden ondersteunt.

Daarna lopen we door de hoofdfeatures heen:

- Declarations
- Classes, modules
- Sets, Maps
- Promises, async/await

Aan het eind van de les een korte vooruitblik naar upcoming standaarden uit de TC39 standaard, zoals Observables, die nu al in RXJS 5 zitten; een dependency van Angular, die standaard meekomt!

Al deze ES features worden ook ondersteunt in TypeScript; zoals natuurlijk de Classes. En TS heeft in versie 1.8+ ook de nieuwe ES modules overgenomen en de eigen module format laten vallen.
*1: Voor sommige features zijn nog wel polyfills nodig.*

Bij de start van de oefening gaan we ook assessen. Je moet nu ook duo's gaan vormen voor App 2. Hier gaan we in de volgende les naar kijken.
